Per eseguire l'applicazione in locale eseguire i seguenti script in ordine all'interno della cartella scaricata (dopo aver estratto il file .zip):
1. Setup_valDB.bat
   Questo script scarica tutte le librerie python necessarie. Crea le tabelle del DB. Poi chiede di valorizzarle con un esempio di contesti.
3. Start_app.bat
   Questo script avvia il backend e il frontend dell'applicazione
